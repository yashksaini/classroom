Create and Join Features

  • Create Class with
    • Class Name
    • Subject

  • Join Class with
    • Class Code ( As shared by the teacher )
    • Teacher Name ( Enter Teacher Name according to your convenience.
      To easily itentify class.)
    • Subject (Enter Subject Name according to your convenience)

    • When user join the class, it sends a request to the class owner to allow the user in the class
    • In the top user can see the requests send for classes.
    • User can withdraw requests for classes that send earliar.

Created and Joined Classes

  • Created Classes
    • Edit Class Name of class
    • Edit Subject
    • Create Assignments
    • Give notice to whole classes
    • Can remove any student from the class
    • Can remove whole class

  • Joined Classes
    • Edit Teacher Name in class
    • Edit Subject
    • Submit Assignments
    • See notices by instructor
    • Can leave class.

  • User can change name from the Profile Section
